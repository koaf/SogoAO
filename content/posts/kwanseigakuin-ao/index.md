関西学院大学の総合型選抜は、一般選抜のように学力試験だけで合否が決まる入試ではありません。

ただし、関西学院大学の総合型選抜には複数の方式があります。

## 1. 関西学院大学の総合型選抜とは

関西学院大学の総合型選抜は、受験生がこれまでに培ってきた基礎学力、活動経験、資質、能力、学ぶ意欲、人間性などを多面的に評価する入試です。

一般選抜では、主に教科試験の得点が重視されます。  
一方で総合型選抜では、書類、活動実績、資格、面接、小論文、プレゼンテーションなどを通して、受験生の考え方や将来性が見られます。

### 総合型選抜で見られやすいポイント

| 評価される力 | 具体的に見られること |
|---|---|
| 目的意識 | なぜ関西学院大学で学びたいのか |
| 探究心 | 高校時代にどんなテーマに取り組んできたか |
| 主体性 | 自分から行動した経験があるか |
| 表現力 | 書類・面接・小論文で考えを伝えられるか |
| 基礎学力 | 入学後の学びに必要な学力があるか |
| 将来性 | 大学での学びを将来にどうつなげたいか |

大事なのは、「すごい実績がないと無理」と考えすぎないことです。

もちろん、資格や大会実績が評価される場合もあります。  
しかし、それ以上に大切なのは、**自分の経験を振り返り、大学で学びたいことと結びつけて説明できるか** です。

## 3. 関西学院大学の総合型選抜は4種類ある

関西学院大学の総合型選抜には、主に4つの方式があります。

| 入試方式 | 活かしやすい経験・強み |
|---|---|
| 探究評価型入学試験 | 高校での探究活動、発表経験 |
| グローバル入学試験 | 国際的な活動、英語資格、国際交流経験 |
| 学部特色入学試験 | 学部に合った経験、英語資格、リーダーシップなど |
| スポーツ選抜入学試験 | 高校入学後のスポーツ実績 |

どの方式も、「関西学院大学で学びたい」という意欲が前提になります。

そのうえで、どの方式が自分に合っているかを考える必要があります。

## 4. 探究評価型入学試験とは

探究評価型入学試験は、高校での探究活動を活かしやすい入試です。

関西学院大学の公式ページでは、探究評価型について、全14学部で実施される入試として紹介されています。

### 探究評価型に向いている人

- 高校の授業で探究活動に取り組んだ
- 課題研究やプロジェクト学習を行った
- 学校内外で探究活動を発表した経験がある
- 自分でテーマを決めて調査・分析・発表した
- 探究活動を大学での学びにつなげたい

探究評価型で重要なのは、単に「探究をやりました」と言うことではありません。

大切なのは、次のような流れを説明できることです。

1. なぜそのテーマに関心を持ったのか
2. どのように調査・活動したのか
3. どんな発見や課題があったのか
4. その経験から何を学んだのか
5. 関西学院大学でどう深めたいのか

### 探究活動の伝え方

| NG | OK |
|---|---|
| 「学校の探究授業で地域課題について調べました」 | 「商店街の来街者減少に関心を持ち、店舗への聞き取りと高校生向けアンケートを行い、若者向けイベントの提案をまとめました」 |
| 「発表を頑張りました」 | 「調査結果をもとに、課題・原因・改善案を整理し、学年発表会で発表しました」 |
| 「環境問題に興味があります」 | 「校内の食品ロス調査を行い、廃棄量の記録と原因分析から、購買方法の改善案を考えました」 |

探究評価型では、立派な研究成果だけでなく、**問いを立て、調べ、考え、伝えたプロセス** が大切です。

## 5. グローバル入学試験とは

グローバル入学試験は、国際的な活動や英語資格を活かしやすい入試です。

「グローバル」と聞くと、長期留学をしていないと出願できないと思う人もいるかもしれません。  
しかし、関西学院大学の公式ページでは、海外留学経験がなくても出願できることが示されています。

### グローバル入学試験に向いている人

- 海外留学や語学研修に参加した
- 国際交流活動に参加した
- 国内で国際交流や多文化交流に関わった
- 英語資格を持っている
- 国際問題や異文化理解に強い関心がある
- 将来、国際的な分野で活動したい

国際的な活動は、海外経験だけではありません。

たとえば、国内での国際交流、地域の外国人支援、多文化共生に関する活動、国際ボランティア、オンライン交流なども、条件に合えばアピール材料になります。

### グローバル系の志望理由で大切なこと

「英語が好きです」だけでは弱くなりやすいです。

次のように、経験と学びたいテーマをつなげましょう。

| NG | OK |
|---|---|
| 「英語が得意なので国際系に進みたいです」 | 「地域の国際交流イベントに参加し、言語だけでなく文化的背景を理解する重要性を感じました。大学では多文化共生について学びたいです」 |
| 「海外に興味があります」 | 「短期語学研修で、同じ社会問題でも国によって考え方が違うことに気づき、国際関係を多角的に学びたいと考えました」 |

英語資格は大切ですが、資格だけで合格が決まるわけではありません。  
**英語や国際経験を通して、何を考え、何を学びたいと思ったのか** を言語化しましょう。

## 6. 学部特色入学試験とは

学部特色入学試験は、学部ごとに求める人物像や出願条件が異なる入試です。

関西学院大学の公式ページでは、学部特色入学試験は12学部で実施されると紹介されています。

### 学部特色入学試験に向いている人

- 受験したい学部がはっきり決まっている
- その学部で学びたい理由を具体的に説明できる
- 英語資格や活動実績がある
- リーダーシップを発揮した経験がある
- 学部の学びと高校時代の経験がつながっている

学部特色入学試験では、「関西学院大学に行きたい」だけでなく、**なぜその学部なのか** が非常に重要です。

たとえば、同じボランティア経験でも、学部によって伝え方は変わります。

| 志望学部 | 経験のつなげ方の例 |
|---|---|
| 社会学部 | 地域やメディア、社会課題への関心につなげる |
| 法学部 | ルール、公共性、制度、社会問題への関心につなげる |
| 経済学部 | 地域経済、格差、消費行動、政策への関心につなげる |
| 商学部 | ビジネス、マーケティング、組織運営への関心につなげる |
| 人間福祉学部 | 福祉、支援、共生社会への関心につなげる |
| 教育学部 | 子ども、学校、教育課題への関心につなげる |
| 総合政策学部 | 地域課題、政策、環境、国際社会への関心につなげる |

学部特色入学試験を受ける人は、まず志望学部のアドミッション・ポリシーを読みましょう。  
そのうえで、自分の経験がどのように学部の学びにつながるかを整理することが大切です。

## 7. スポーツ選抜入学試験とは

スポーツ選抜入学試験は、高校入学後のスポーツ活動において、優れた能力や競技実績を持つ人を対象とした入試です。

関西学院大学の公式ページでは、スポーツ選抜は全14学部で実施されるとされています。

### スポーツ選抜に向いている人

- 高校入学後に競技実績がある
- 部活動やクラブチームで継続的に競技に取り組んだ
- 競技力だけでなく、学業にも取り組む意欲がある
- 大学入学後も競技やチーム活動に貢献したい
- スポーツ経験を将来の学びや進路につなげたい

スポーツ選抜では、競技実績が重要です。  
ただし、面接や書類では、競技結果だけでなく、そこから何を学んだかも問われやすいです。

たとえば、

- 目標達成のためにどう努力したか
- ケガや不調をどう乗り越えたか
- チームの中でどのような役割を果たしたか
- 競技経験を大学でどう活かしたいか
- 学業と競技をどう両立したいか

を整理しておきましょう。

## 8. 併願に関する重要な注意点

関西学院大学の総合型選抜では、併願について特に注意が必要です。

### 総合型選抜内での併願はできない

関西学院大学の総合型選抜では、同一年度内に複数の総合型選抜へ出願することはできません。

つまり、たとえば、

- 探究評価型とグローバル入学試験を両方受ける
- 学部特色入学試験で複数の学部に出願する
- 同じ年度内に別の総合型選抜へ再出願する

といったことはできません。

出願できるのは、**1つの入学試験・1つの学部・学科・課程・専修・コースのみ** です。

### 合格した場合は入学が条件

関西学院大学の総合型選抜は、関西学院大学を第一志望とする受験生を対象としています。

そのため、合格した場合は入学することが条件です。

> 注意：総合型選抜は「とりあえず受けておく入試」ではありません。出願前に、本当に関西学院大学で学びたいのか、志望学部で何を学びたいのかをしっかり確認しましょう。

### 不合格後の一般選抜は可能

総合型選抜で不合格になった場合でも、関西学院大学の一般選抜に出願することは可能です。

ただし、同一年度内に別の総合型選抜へ出願することはできません。

## 9. 関西学院大学の総合型選抜に向けた準備

関西学院大学の総合型選抜では、方式ごとに準備内容が変わります。  
まずは、自分がどの方式に向いているかを考えましょう。

### 方式別・準備の方向性

| 方式 | まず準備したいこと |
|---|---|
| 探究評価型 | 探究活動のテーマ・方法・成果・学びを整理する |
| グローバル | 国際的な活動経験と英語資格を確認する |
| 学部特色 | 志望学部の学びと自分の経験を結びつける |
| スポーツ選抜 | 競技実績、役割、努力の過程を整理する |

### 志望理由書で意識したい流れ

志望理由書では、次の流れを意識すると書きやすくなります。

1. 関心を持ったきっかけ
2. 高校時代に取り組んだこと
3. その経験から見えた課題や学び
4. 関西学院大学の志望学部で学びたいこと
5. 入学後に取り組みたいこと
6. 将来どのように活かしたいか

「関学に入りたい」という気持ちは大切です。  
でも、それだけでは志望理由としては弱くなります。

大切なのは、**自分の経験と、関西学院大学での学びをつなげること** です。

## 10. 面接対策で準備したい質問

総合型選抜では、面接審査が行われる場合があります。  
面接には、口頭試問、プレゼンテーション、グループディスカッションなどが含まれる場合もあります。

### よく準備しておきたい質問

- なぜ関西学院大学を志望したのですか？
- なぜその学部を選んだのですか？
- この入試方式を選んだ理由は何ですか？
- 高校時代に最も力を入れた活動は何ですか？
- その活動で大変だったことは何ですか？
- 失敗した経験から何を学びましたか？
- 入学後に学びたいテーマは何ですか？
- 将来はどのような進路を考えていますか？
- 最近気になっている社会問題は何ですか？
- あなたの強みを大学生活でどう活かしますか？

丸暗記する必要はありません。  
ただし、どの質問にも **経験 → 気づき → 学びたいこと** の流れで答えられるようにしておきましょう。

## 11. よくある失敗と注意点

### 失敗1：方式選びをなんとなく決める

関西学院大学の総合型選抜は、方式によって評価される経験が違います。

探究活動を活かしたいのか、国際経験を活かしたいのか、学部の特色に合った経験を伝えたいのか。  
まずは自分の強みを整理してから方式を選びましょう。

### 失敗2：証明資料の準備が遅れる

活動実績や資格をアピールする場合、客観的な資料が必要になることがあります。

たとえば、

- 賞状
- 資格証明書
- 活動証明書
- 発表資料
- 顧問や担任による証明
- 新聞記事や活動記録

などです。

資料がない場合、評価対象にならない可能性もあります。  
早めに学校の先生へ相談しておきましょう。

### 失敗3：志望理由が抽象的すぎる

「校風に惹かれました」「有名だからです」「国際的な雰囲気が良いと思いました」だけでは弱いです。

次のように、自分の経験と学部の学びをつなげましょう。

| NG | OK |
|---|---|
| 「関西学院大学の雰囲気が好きです」 | 「探究活動で地域の外国人支援について調べた経験から、多文化共生について学びたいと考えました」 |
| 「国際的な大学なので志望しました」 | 「国際交流活動を通じて、文化の違いによる誤解に関心を持ち、国際社会におけるコミュニケーションを学びたいと考えました」 |
| 「リーダーシップがあります」 | 「部活動で練習メニューの改善を提案し、チーム内で役割分担を見直した経験があります」 |

### 失敗4：第一志望である理由を説明できない

関西学院大学の総合型選抜は、第一志望であることが前提です。

だからこそ、面接や書類では、

- なぜ関西学院大学なのか
- なぜその学部なのか
- 入学後に何をしたいのか

を具体的に説明できるようにしておく必要があります。

面接で聞かれたこと、志望理由書で工夫したこと、小論文やプレゼンの内容、準備してよかったことなどは、これから受験する高校生にとって大きな助けになります。

## 【まとめ】関西学院大学の総合型選抜は「自分の経験に合う方式選び」が重要

関西学院大学の総合型選抜には、探究評価型、グローバル、学部特色、スポーツ選抜の4つの方式があります。

それぞれ評価される経験や強みが異なるため、まずは自分が高校時代に取り組んできたことを整理しましょう。

- [ ] 探究活動を活かしたいなら、探究評価型
- [ ] 国際的な活動や英語資格を活かしたいなら、グローバル入学試験
- [ ] 志望学部に合った経験を伝えたいなら、学部特色入学試験
- [ ] スポーツ実績を活かしたいなら、スポーツ選抜入学試験

総合型選抜は、早めに準備した人ほど自分の経験を深く整理できます。

まずは公式情報を確認し、出願資格、提出書類、選考内容をチェックするところから始めましょう。
