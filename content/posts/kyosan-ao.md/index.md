## 1. 京都産業大学の総合型選抜入試とは

京都産業大学の総合型選抜入試は、従来のペーパーテストや偏差値だけでは測れない、多様な可能性をもつ受験生を評価する入試です。

大学公式ページでは、総合型選抜入試で問われるものとして、次のような力が重視されています。

- 積極性
- 問題意識
- 目的意識
- 知的探究心
- 入学後に成長する可能性

一般選抜のように、教科試験の点数だけで合否が決まる入試ではありません。  
出願書類、講義理解、筆記試験、小論文、プレゼンテーション、面接、グループディスカッション、作品審査などを通して、受験生の考え方や学ぶ意欲が見られます。

### 総合型選抜で見られやすいポイント

| 評価される力 | 具体的に見られること |
|---|---|
| 目的意識 | なぜ京都産業大学で学びたいのか |
| 問題意識 | 社会や学問分野にどんな関心を持っているか |
| 表現力 | 志望理由書・小論文・プレゼン・面接で伝えられるか |
| 主体性 | 高校時代にどのような活動や探究をしてきたか |
| 学習意欲 | 入学後に何を学び、どう成長したいか |
| 基礎学力 | 講義理解、筆記試験、口頭試問などに対応できるか |

大事なのは、「すごい実績があるか」だけではありません。  
むしろ、**自分の経験をどう振り返り、大学での学びにつなげられるか** が重要です。

## 3. 出願資格と注意したいポイント

京都産業大学の総合型選抜入試では、大学で学びたいという強い意志と能力を備えていることが前提です。

主な出願資格は、次のいずれかに該当する人です。

- 高等学校または中等教育学校を卒業した人、または2027年3月31日までに卒業見込みの人
- 通常の課程による12年の学校教育を修了した人、または修了見込みの人
- 高等学校卒業と同等以上の学力があると認められる人

ただし、ここで注意したいのは、**学部によって追加の出願要件がある** ことです。

たとえば、経営学部、現代社会学部、外国語学部、文化学部、理学部の一部学科、情報理工学部、生命科学部などでは、通常の出願資格に加えて、学部・学科ごとの条件を満たす必要があります。

### 追加要件の例

| 学部 | 注意したい出願要件の例 |
|---|---|
| 経営学部 | マネジメントに関連した経験や実績が必要 |
| 現代社会学部 | Aタイプ・Bタイプがあり、Bタイプでは学習成績の状況などの条件あり |
| 外国語学部 | 語学資格、研究テーマ、特色ある活動などのいずれかが必要 |
| 文化学部 | 文化に関する活動、英語資格、文化系検定などが関係 |
| 理学部 | 数学・物理などの履修条件や資格条件あり |
| 情報理工学部 | 自作の作品と説明資料の提出が必要 |
| 生命科学部 | 課題提出型または学科試験型を選択 |
| アントレプレナーシップ学環 | 志望理由書に加え、自己アピール動画の提出が必要 |

出願要件を満たしていないと、どれだけ志望理由が良くても出願できません。  
最初に確認するべきなのは、「自分はこの学部に出願できるか」です。

## 5. 学部別に見た対策ポイント

ここからは、学部ごとにどんな準備が必要になりやすいかを整理します。

### 経済学部：社会問題への関心を自分の言葉で説明する

経済学部では、1次選考で模擬講義を受け、その内容に関する記述式テストがあります。  
2次選考では、自分が関心を持った社会問題について5分程度で発表し、その後に質疑応答があります。

ポイントは、経済問題に限定されないことです。  
ただし、報道や本の内容をそのまま紹介するだけでは不十分です。

自分なりに、

- なぜその問題に関心を持ったのか
- 何が課題だと考えるのか
- どのような背景があるのか
- 自分はどう考えるのか

を整理しておきましょう。

### 経営学部：経験を「マネジメント」と結びつける

経営学部のマネジメント力選抜入試では、高校在学中にマネジメントに関連した経験があることが求められます。

ここでいうマネジメントは、会社経営だけではありません。

たとえば、

- 部活動でチーム運営に関わった
- 文化祭や行事で企画・調整をした
- ボランティア活動で役割分担を考えた
- コンテストやプロジェクトで人をまとめた
- 起業やビジネスに関心を持って活動した

といった経験も、伝え方によってはマネジメントの経験になります。

大事なのは、「何をしたか」だけでなく、**どんな課題に気づき、どう工夫し、何を学んだか** です。

### 法学部：社会問題を調べ、論理的に説明する

法学部では、1次選考で小論文、2次選考でプレゼンテーションがあります。

プレゼンでは、志望理由を述べたうえで、自分で設定した社会問題について、現状・課題・改善策を発表します。

法学部を受ける場合は、ニュースを見て「大変そう」で終わらせるのではなく、

- どのような社会問題か
- なぜその問題が起きているのか
- 誰にどのような影響があるのか
- どんなルールや制度が関係しているのか
- どのような改善策が考えられるのか

まで考える練習が必要です。

### 国際関係学部：国際問題を多角的に考える

国際関係学部では、1次選考で国際問題に関する講義と記述式テストがあります。  
2次選考では、グループディスカッションと個人面接があります。

国際関係学部を受ける人は、英語力だけでなく、世界で起きている出来事に対する関心や、自分とは異なる価値観を理解する姿勢が大切です。

日ごろから、国際ニュースについて、

- その問題の背景
- 関係する国や地域
- 立場の違い
- 解決が難しい理由
- 自分ならどう考えるか

をメモしておくと、面接やグループディスカッションの準備になります。

### 外国語学部：語学力だけでなく、学びたいテーマを明確にする

外国語学部では、1次選考が書類審査、2次選考が小論文と個人面接です。

英語学科、ヨーロッパ言語学科、アジア言語学科など、専攻によって見られるポイントは異なります。  
ただし共通して大事なのは、単に「語学が好きです」で終わらせないことです。

- なぜその言語・地域に関心を持ったのか
- その国や地域の文化・社会について何を調べたのか
- 大学でどのように学びを深めたいのか
- 将来、語学をどう活かしたいのか

を具体的に説明できるようにしましょう。

### 文化学部：文化・観光・発信への関心を研究テーマにする

文化学部では、文化に関する経験や実績、英語資格、文化系検定などが出願要件に関係します。  
1次選考ではOne Dayセミナー、2次選考ではプレゼンテーションと面接があります。

文化学部で大事なのは、「文化が好き」から一歩進めて、**何を研究テーマにしたいのか** を言語化することです。

たとえば、

- 地域文化の継承
- 京都文化の発信
- 観光と地域社会の関係
- デジタルメディアと文化発信
- 文学・芸術・思想への関心
- 文化観光の課題

など、自分の経験や関心と学部の学びをつなげる必要があります。

### 理学部：レポート・筆記・口頭試問への準備が必要

理学部では、学科によって選考内容が異なります。

数理科学科では、課題レポートや数学Ⅲの筆記試験、プレゼンテーション、数学の基礎知識に関する質疑応答があります。  
物理科学科では、数学・物理の筆記試験と、面接内での口頭試問があります。  
宇宙物理・気象学科では、課題レポートに基づくプレゼンと、物理学・数学の基礎知識に関する口頭試問があります。

理学部を受ける場合は、活動実績だけでなく、**基礎学力を説明できる力** が必要です。

### 情報理工学部：作品の完成度より「説明できること」が重要

情報理工学部では、自分で作成したハードウェアやソフトウェアなどの作品を提出します。  
1次選考では作品と説明書を含む書類審査、2次選考では作品に関する個人面接があります。

ここで大事なのは、作品が派手かどうかだけではありません。

- なぜその作品を作ったのか
- どんな課題を解決したかったのか
- どの部分を自分で作ったのか
- AIや他者の助力を使った場合、どこまで自分が理解しているのか
- 工夫した点は何か
- 改善するとしたら何を変えるか

を説明できることが大切です。

### 生命科学部：課題提出型か学科試験型かを早めに決める

生命科学部では、課題提出型と学科試験型のどちらかを選びます。

課題提出型では、生物や化学の課題研究・探究活動の成果をまとめた成果報告書が重要になります。  
学科試験型では、生物基礎または化学基礎の学科試験、模擬講義、振り返りレポートが行われます。

探究活動の実績がある人は課題提出型が向いている可能性があります。  
一方で、理科の基礎学力をしっかり示したい人は学科試験型も選択肢になります。

### アントレプレナーシップ学環：挑戦の過程を語れるかが大事

アントレプレナーシップ学環の探究・挑戦志向入試では、志望理由書に加えて、3分以内の自己アピール動画を提出します。

ここで見られるのは、成功体験だけではありません。  
むしろ、挑戦した過程や、うまくいかなかった経験から何を学んだかも重要です。

- どんなテーマに取り組んだのか
- なぜそのテーマに挑戦したのか
- どんな行動をしたのか
- 結果から何を学んだのか
- 入学後、事業や社会課題解決にどうつなげたいのか

を一貫して伝える準備をしましょう。

## 6. 志望理由書・面接・プレゼンの対策

京都産業大学の総合型選抜では、多くの学部で志望理由書が重要になります。

志望理由書は、単に「入学したいです」と伝える書類ではありません。  
自分の経験、問題意識、学びたい内容、将来像をつなげる書類です。

### 志望理由書で意識したい流れ

| 順番 | 書く内容 |
|---|---|
| 1 | 自分が関心を持ったきっかけ |
| 2 | 高校時代に取り組んだこと |
| 3 | そこから見えてきた課題や問題意識 |
| 4 | 京都産業大学のその学部で学びたい理由 |
| 5 | 入学後に取り組みたいこと |
| 6 | 将来どのように活かしたいか |

### NG例とOK例

| NG | OK |
|---|---|
| 「貴学は有名で、雰囲気が良いと思ったからです」 | 「高校で地域の観光課題について調査した経験から、文化観光を学び、地域の魅力発信について研究したいと考えました」 |
| 「昔から国際関係に興味があります」 | 「ニュースで難民問題を知り、支援する側と受け入れる側の立場の違いに関心を持ちました。大学では国際協力と地域社会の関係を学びたいです」 |
| 「リーダーシップがあります」 | 「文化祭実行委員として意見が分かれた場面で、役割分担表を作り、全員が動きやすい形に調整しました」 |
| 「情報理工学部でプログラミングを学びたいです」 | 「部活動の予定共有を効率化するWebアプリを作った経験から、情報技術で身近な課題を解決する力を伸ばしたいです」 |

大事なのは、抽象的な言葉を並べることではありません。  
**自分の経験を具体的に書くこと** です。

### 面接対策で準備したい質問

面接では、志望理由書に書いた内容をもとに質問されることが多くあります。

最低限、次の質問には答えられるようにしておきましょう。

- なぜ京都産業大学を志望したのですか？
- なぜその学部・学科なのですか？
- 高校時代に力を入れた活動は何ですか？
- その活動で大変だったことは何ですか？
- 失敗した経験から何を学びましたか？
- 入学後に学びたいテーマは何ですか？
- 卒業後はどのような進路を考えていますか？
- 最近気になっている社会問題は何ですか？
- あなたの強みは何ですか？
- その強みを大学生活でどう活かしますか？

丸暗記する必要はありません。  
ただし、どの質問にも **経験 → 気づき → 大学での学び** の流れで答えられるようにしておくと、話に一貫性が出ます。

### プレゼン対策で意識したいこと

プレゼンテーションがある学部では、資料の見た目よりも、内容の筋道が大切です。

おすすめの構成は次の通りです。

1. テーマを選んだ理由
2. 現状・背景
3. 調べてわかった課題
4. 自分の考え・提案
5. 京都産業大学で学びたいこと
6. 将来につなげたいこと

プレゼンでは、調べた情報をそのまま読むだけでは弱くなります。  
資料をもとにしながらも、最後は **自分の考え** を示すことが重要です。

## 7. よくある失敗と注意点

### 失敗1：出願要件を後から確認する

総合型選抜では、学部によって追加の出願要件があります。  
資格、履修科目、活動実績、作品、成果報告書などが必要になる場合があります。

志望理由書を書き始める前に、まずは出願要件を確認しましょう。

### 失敗2：志望理由が「大学名」だけになっている

「京都産業大学に行きたい」だけでは不十分です。

大事なのは、

- なぜその学部なのか
- 何を学びたいのか
- 自分の経験とどうつながるのか
- 入学後にどう成長したいのか

まで説明することです。

### 失敗3：活動実績をただ並べるだけ

総合型選抜では、活動実績の量だけで決まるわけではありません。

たとえ大きな賞がなくても、

- 自分で課題を見つけた
- 工夫して行動した
- 周囲と協力した
- 失敗から改善した
- 次の学びにつなげた

という過程を説明できれば、十分にアピールになります。

### 失敗4：面接で書類と違うことを話してしまう

面接では、志望理由書や提出書類の内容について聞かれることがあります。  
自分で書いた内容を説明できないと、準備不足に見えてしまいます。

提出前に必ずコピーを取り、面接前に読み返しましょう。

### 失敗5：公式情報を確認しない

入試日程、出願条件、提出書類、試験内容は変更される可能性があります。  
必ず京都産業大学の公式サイトと最新の入学試験要項を確認してください。

## 【まとめ】京都産業大学の総合型選抜は「学部別対策」が重要

京都産業大学の総合型選抜入試は、学部によって選考内容が大きく異なります。

経済学部や文化学部のようにOne Dayセミナーがある学部もあれば、法学部のように小論文とプレゼンが重視される学部もあります。  
情報理工学部では作品提出、生命科学部では課題提出型・学科試験型、アントレプレナーシップ学環では自己アピール動画が必要です。

そのため、まずやるべきことは次の3つです。

- [ ] 自分が受けたい学部の出願要件を確認する
- [ ] 選考方法に合わせて、書類・小論文・面接・プレゼン対策を始める
- [ ] 自分の経験と大学で学びたいことをつなげて言語化する

総合型選抜は、早く準備を始めた人ほど有利になりやすい入試です。  
まずは公式情報を確認し、自分が受ける学部では何が求められているのかを整理するところから始めましょう。
